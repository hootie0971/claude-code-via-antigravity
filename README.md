## ⚙️ What is claude-code-via-antigravity?

This app lets you use the Claude Code command line tool (CLI) for free through your Google Antigravity or Code Assist subscription. You don’t need an Anthropic API key. It works on Linux, macOS, and Windows computers.

## 🔍 How to Use Claude Code CLI

This app works primarily from the command line, which means you type simple commands to interact with it.

---

### Opening your command line tool:

- **Windows:** Press Windows key, type `cmd`, and open Command Prompt.
- **macOS:** Open Finder, go to Applications > Utilities, and open Terminal.
- **Linux:** Open Terminal from your application menu or press Ctrl+Alt+T.

---

### Running a command

Here’s a simple way to start using Claude Code:

1. Open your terminal.
2. Type: `claude-code --help` and press Enter.

This shows you all the commands you can use.

---

### Basic commands to try

- `claude-code write "Your idea here"`  
  This tells Claude to help you write code based on what you describe.

- `claude-code debug /path/to/your/code`  
  This helps identify problems in your code files.

- `claude-code explain "Code snippet here"`  
  Claude explains what a piece of code does in plain English.
